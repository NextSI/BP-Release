# Correção na descrição de campos do tipo data do formulário do processo
29/12/2020
## Correção
## Referência: CHM49068
## Módulo: Processo
***

Esta versão corrige o problema que gravava os valores de campo tipo data na descrição do processo no formato yyyy-mm-dd para dd/mm/yyyy porém este correção não resolve os registros antigos já salvos na base de dados.
